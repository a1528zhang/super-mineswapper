# 超级扫雷

一个微信小游戏版的基础扫雷项目，当前实现为 `9x9` 棋盘，支持长按插旗、双击数字自动展开。

## 目录结构

### 根目录
- `game.js`：小游戏入口文件，负责初始化画布、绑定输入事件、协调渲染与游戏逻辑。
- `game.json`：微信小游戏基础配置。
- `project.config.json`：微信开发者工具项目配置。
- `project.private.config.json`：微信开发者工具私有配置。
- `super-minesweeper-icon.png`：项目图标资源。
- `README.md`：项目说明文档。

### `js/config`
- `game-config.js`：游戏常量配置，如棋盘尺寸、雷数、长按时间、双击时间、游戏状态枚举。

### `js/models`
- `cell.js`：方块数据模型，定义每个格子的基础结构。

### `js/logic`
- `board.js`：棋盘相关规则。
  - 创建棋盘
  - 布雷
  - 计算周围雷数
  - 递归展开空白区域
  - 显示所有雷
- `game-controller.js`：游戏控制器，负责处理翻格、插旗、双击数字展开、胜负判断等核心流程。
- `victory.js`：胜利相关规则，如是否获胜、获胜后自动标记剩余雷。

### `js/input`
- `touch-controller.js`：输入事件处理。
  - 单击翻开
  - 长按插旗
  - 双击数字触发展开
  - 重开按钮点击

### `js/ui`
- `renderer.js`：整体界面渲染入口，负责标题、状态、按钮和棋盘绘制。
- `cell-view.js`：单个格子的绘制逻辑，包括未翻开、已翻开、旗帜、地雷、数字。
- `layout.js`：界面布局计算，以及坐标转格子坐标。
- `draw-utils.js`：通用绘制工具，如圆角矩形、区域判断。
- `status-text.js`：状态提示文案。

## 当前功能

- `9x9` 扫雷棋盘
- 首次点击不会踩雷
- 长按插旗 / 取消插旗
- 双击已展开数字，满足条件时自动展开周边未插旗格子
- 游戏胜利与失败状态提示
- 重开按钮

## 运行方式

在微信开发者工具中打开本目录即可运行。
