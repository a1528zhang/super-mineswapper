# 超级扫雷

一个微信小游戏版的基础扫雷项目，初始为 `10x10` 棋盘，支持长按插旗、双击数字自动展开，并加入了定时增长行的生存玩法。

## 目录结构

### 根目录
- `game.js`：小游戏入口文件，负责初始化画布、绑定输入事件、协调渲染与游戏逻辑。
- `game.json`：微信小游戏基础配置。
- `project.config.json`：微信开发者工具项目配置。
- `project.private.config.json`：微信开发者工具私有配置。
- `super-minesweeper-icon.png`：项目图标资源。
- `README.md`：项目说明文档。

### `js/config`
- `game-config.js`：游戏常量配置，如棋盘尺寸、雷数、长按时间、双击时间、游戏状态枚举。

### `js/models`
- `cell.js`：方块数据模型，定义每个格子的基础结构。

### `js/logic`
- `board.js`：棋盘相关规则。
  - 创建棋盘
  - 布雷
  - 计算周围雷数
  - 递归展开空白区域
  - 显示所有雷
- `game-controller.js`：游戏控制器，负责处理翻格、插旗、双击数字展开、胜负判断等核心流程。
- `board.js` 还负责动态新增行、行消除、重新计算邻雷数等地图变化逻辑。
- `victory.js`：胜利相关规则，如是否获胜、获胜后自动标记剩余雷。

### `js/input`
- `touch-controller.js`：输入事件处理。
  - 单击翻开
  - 长按插旗
  - 双击数字触发展开
  - 重开按钮点击

### `js/ui`
- `renderer.js`：整体界面渲染入口，负责标题、状态、按钮和棋盘绘制。
- `cell-view.js`：单个格子的绘制逻辑，包括未翻开、已翻开、旗帜、地雷、数字。
- `layout.js`：界面布局计算，以及坐标转格子坐标。
- `draw-utils.js`：通用绘制工具，如圆角矩形、区域判断。
- `status-text.js`：状态提示文案。

## 当前功能

- 初始 `10x10` 扫雷棋盘
- 游戏开始后倒计时 3 分钟，时间为 0 时游戏停止
- 首次点击不会踩雷
- 长按插旗 / 取消插旗，也可以通过插旗模式按钮快速切换点击插旗
- 双击已展开数字，满足条件时自动展开周边未插旗格子
- 每次点击或双击数字触发行消除时计分：消除 1 行得 1 分，2 行得 3 分，3 行得 5 分，超过 3 行得 `5 + 行数 * 2` 分
- 每次展开砖块后，任意未展开且未插旗的砖块都有机会出现时间奖励；10 秒内展开且不是雷时剩余时间增加 10 秒，超过 10 秒会变回普通格子
- 初始每过 10 秒顶部新增一行砖块，每过 30 秒新增间隔减少 1 秒，最低减少至 3 秒
- 当前棋盘低于 5 行时会立即补充新增行
- 一行中所有安全格都展开，且剩余插旗格都是真雷时，该行会渐隐消失，下方行平滑上移补位
- 地图高度达到 `20x10` 时失败，并在第 20 行位置显示死线
- 游戏胜利与失败状态提示
- 游戏结束时展示最终分数
- 雷区上方显示剩余时间，格式为时分秒
- 重开按钮

## 运行方式

在微信开发者工具中打开本目录即可运行。
